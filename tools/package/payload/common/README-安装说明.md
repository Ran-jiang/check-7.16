# CCiteheck 法律引注核查 — 安装说明

本安装包自带全部运行环境（Python、Node、法规库、EUR-Lex 本地服务），无需预装任何软件。
安装后本机后台常驻两个服务（仅监听本机 127.0.0.1）：核查 API（3000，HTTPS）与 EUR-Lex（3010）。

## macOS 安装

1. 解压 zip；
2. **右键点击 `install.command` → 打开**（首次会提示"无法验证开发者"，右键打开即可放行；直接双击会被系统拦截）；
3. 过程中会弹出钥匙串授权（安装 localhost HTTPS 证书），请输入登录密码并选"始终允许"；
4. 安装器最后会自动打开网页版 https://localhost:3000 并显示自检结果；
5. 打开 Word → 开始 → 加载项 → **CCiteheck 法律引用核查**。

卸载：运行 `~/Applications/CCiteheck/uninstall.command`。

## Windows 安装

1. 解压 zip；
2. 双击 `install.bat`（SmartScreen 提示时点"更多信息 → 仍要运行"；会自动请求管理员权限——用于信任 HTTPS 证书和注册 Word 加载项目录）；
3. 安装器最后会自动打开网页版并显示自检结果；
4. 打开 Word → 开始 → 加载项 → 高级 → **共享文件夹** → CCiteheck 法律引用核查 → 添加。

卸载：以管理员身份运行 `%LOCALAPPDATA%\CCiteheck\uninstall.ps1`。

## 使用

- **Word 插件**：核查全文 / 核查选中内容；点结果卡片"定位原文"跳转；对结论做接受/忽略标记。
- **网页版**：https://localhost:3000 ，上传 DOCX 直接核查。
- 核查以法规**现行有效版本**为基准；文书引用历史版本时请自行核对行为时文本。

## 常见问题

- **服务没起来 / 网页打不开**：看日志——mac：`~/Applications/CCiteheck/logs/`；win：`%LOCALAPPDATA%\CCiteheck\logs\`。
- **Word 里看不到加载项**：先完全退出 Word 再重开；mac 可重跑一遍安装器（会清理 Word 加载项缓存）。
- **端口被占**：3000/3010 被其他程序占用时安装器会明确提示，释放后重装。
- **自检显示某数据源未配置**：`.env` 里对应密钥为空。本包若为内部分发版已内置密钥；否则请联系分发人获取。
- **浏览器提示证书不受信任 / 到期**：本机 HTTPS 证书有效期约一年，到期后重新运行安装器即可自动续期。

## 安全须知（内部分发版）

`.env` 内含千问/北大法宝密钥明文，**请勿把安装包转发到公司外或上传公网**；密钥如疑似泄露请联系分发人轮换。
